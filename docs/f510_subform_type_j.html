【種別サブフォーム】

レイアウト

初期表示

  • 月次タブで選択された年月と種別で支出情報を絞り込む．
  • 絞り込んだ支出情報を支出テーブルへ表示する．
  • 絞り込んだ支出情報が0件の場合，支出テーブルは0行とする．

ツールチップの表示

  • 備考のある行にマウスオンしたとき，ツールチップを表示する．
  • ツールチップには支出情報の備考を表示する．
  • 備考のある行は背景色を変更する(#DEFCE7)．
